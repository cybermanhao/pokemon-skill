# 特性查询

## 概述

特性（Abilities）是宝可梦的天赋能力，每只精灵通常有1-3个特性，部分精灵有隐藏特性。

## 数据来源

- **PokéAPI**: `https://pokeapi.co/api/v2/ability/{name}`
- **PokemonDB**: `https://pokemondb.net/ability/{name}`

## 特性分类

### 战斗特性

| 分类 | 说明 | 例子 |
|------|------|------|
| 攻击型 | 造成额外效果 | 舍身、胆量 |
| 防御型 | 减少伤害 | 耐热、硬壳 |
| 辅助型 | 队伍增益 | 恶作剧之心、接球手 |
| 特殊型 | 特殊效果 | 静电、威吓 |

### 非战斗特性

| 特性 | 效果 |
|------|------|
| 颜色 | 决定肤色 |
| 慢启动 | 出门前5回合速度/攻击减半 |
| 斗争心 | 异性伤害加成 |

## 常见特性效果

### 攻击型

| 特性 | 效果 | 代表精灵 |
|------|------|----------|
| 舍身 | 招式反伤害x1.5 | 大岩蛇、圈圈熊 |
| 胆量 | 同属性技能威力x1.5 | 火焰马、炽焰咆哮虎 |
| 负电 | 对方负电时特攻+20% | 负电拍拍 |
| 正电 | 对方正电时特攻+20% | 正电拍拍 |
| 毛皮大衣 | 受到的物理伤害x0.5 | 穿山王、毛毛角兽 |

### 辅助型

| 特性 | 效果 | 代表精灵 |
|------|------|----------|
| 恶作剧之心 | 变化技能优先度+1 | 滑滑小子、乌贼王 |
| 接球手 | 队友使用变化技能必定成功 | 哥德小姐 |
| 踩影 | 对手无法逃跑 | 果然翁、图腾鸟 |
| 威吓 | 对方攻击-1 | 大奶罐、暴鲤龙 |
| 压迫感 | 对方特攻-1 | 大岩蛇、卡比兽 |

### 特殊型

| 特性 | 效果 | 代表精灵 |
|------|------|----------|
| 静电 | 30%几率麻痹接触技 | 皮卡丘、雷伊布 |
| 火焰皮肤 | 接触技附加30%灼伤 | 火焰马 |
| 蓄电 | 受到电属性伤害时回血 | 雷伊布、皮卡丘 |
| 降雨 | 进场时天气变为雨 | 大甲、化石盔 |
| 茂盛 | HP<1/3时草属性技能威力x1.5 | 妙蛙种子、蜥蜴王 |

### 隐藏特性 (HA)

隐藏特性只能通过特定方式获得（如生蛋、霸主、幻兽等）。

| 特性 | 效果 | 精灵 |
|------|------|------|
| 适应力 | 同属性技能威力x2 | 黏黏宝、铝钢龙 |
| 性能 | 装备道具效果x1.5 | 卡蒂狗、风速狗 |
| 多元族 | 属性变为弱点x0.5 | 谜拟Q |
| 复制 | 复制对手特性 | 谜拟Q |

## 特性强度分级

### T0 (顶级)

- **威吓** - 种族级特性
- **恶作剧之心** - 队伍辅助核心

### T1 (强)

- **静电/火焰皮肤/蓄电** - 状态附加
- **茂盛/猛火/激流** - 低血量加成

### T2 (可用)

- **舍身/胆量** - 输出加成
- **压迫感** - 降攻debuff

### T3 (弱)

- 多数普通特性

## 特性获取方式

| 方式 | 说明 |
|------|------|
| 野生 | 野生精灵通常有普通特性 |
| 蛋技能 | 通过生蛋遗传获得特定特性 |
| 霸主 | 霸主精灵必有隐藏特性 |
| 幻兽/传说 | 传说精灵有特定特性 |

## 查询技巧

- 特性列表：https://pokeapi.co/api/v2/ability?limit=1000
- 按精灵查询：查询精灵数据中的abilities字段
- 按效果查询：筛选特定效果的特性

## 战斗要点

1. **特性配合**：选择与技能/道具配合的特性
2. **队伍协调**：考虑特性对队伍的整体影响
3. **隐藏特性价值**：部分隐藏特性显著强于普通特性
