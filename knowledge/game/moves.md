# 技能查询

## 概述

技能（Moves）是宝可梦在战斗中使用的能力，属性、威力、PP、效果各不相同。

## 数据来源

- **PokéAPI**: `https://pokeapi.co/api/v2/move/{name}`
- **PokemonDB**: `https://pokemondb.net/move/{name}`

## 技能数据字段

| 字段 | 含义 | 示例 |
|------|------|------|
| name | 技能名 | flamethrower |
| type | 属性 | fire |
| power | 威力 | 90 |
| accuracy | 命中率 | 100 |
| pp | PP值 | 20 |
| category | 分类 (物理/特殊/变化) | special |
| effect_chance | 附加效果概率 | 10 |
| description | 效果说明 | 10%几率灼伤 |

## 技能分类

### 按战斗效果

| 分类 | 说明 | 例子 |
|------|------|------|
| 物理 | 使用Atk计算伤害 | 撞击、攀爬 |

## 技能效果

### 状态附加

| 状态 | 概率 | 技能 |
|------|------|------|
| 灼伤 | 10% | 喷射火焰、热风 |
| 冰冻 | 10% | 冰冻光束、绝对零度 |
| 睡眠 | 20% | 睡眠粉、剧毒 |
| 麻痹 | 30% | 十万伏特、雷电打击 |
| 中毒 | 30% | 毒击、污泥炸弹 |

### 变化效果

| 效果 | 技能 |
|------|------|
| 提升攻击 | 龙舞、剑舞 |
| 提升防御 | 金属音、变硬 |
| 降低对方攻击 | 鬼火、毒菱 |
| 回复 | 睡觉、寄生种子 |

## PP与最大值

- PP上限：32（通过PP最大值提升道具）
- 技能机/威力教up：PP翻倍
- PP耗尽后需要回复或更换精灵

## 高威力技能

| 技能 | 威力 | 副作用 |
|------|------|--------|
| 破坏光线 | 150 | 下一回合无法行动 |

## 常用查询

- 技能列表：https://pokeapi.co/api/v2/move?limit=1000
- 属性技能：https://pokeapi.co/api/v2/type/{type}/moves
- 技能机：技能机编号对应技能

## 战斗技巧

1. **属性一致加成 (STAB)**：使用同属性技能伤害 x1.5
2. **技能机复用**：部分技能可以传给其他精灵
3. **先制技能**：优先度高的技能可抢先攻击
