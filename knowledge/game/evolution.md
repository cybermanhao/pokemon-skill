# 进化链

## 概述

宝可梦通过进化从幼年形态成长为更强形态，进化条件各不相同。

## 数据来源

- **PokéAPI**: `https://pokeapi.co/api/v2/evolution-chain/{id}`
- **PokemonDB**: `https://pokemondb.net/evolution/`

## 进化条件分类

### 等级进化

最常见的进化方式，等级达到后自动进化。

| 精灵 | 进化等级 | 进化后 |
|------|----------|--------|
| 妙蛙种子 | 16 | 妙蛙草 |
| 小火龙 | 16 | 火恐龙 |
| 杰尼龟 | 16 | 卡咪龟 |
| 绿毛虫 | 7 | 铁甲蛹 |
| 铁甲蛹 | 10 | 巴大蝶 |
| 波波 | 18 | 比比鸟 |
| 小拉达 | 20 | 拉达 |

### 亲密度进化

| 精灵 | 条件 | 进化后 |
|------|------|--------|

| 小福蛋 | 亲密度220 | 吉利蛋 |

## 进化链结构

### 单阶段进化

```
小火龙 → 火恐龙 → 喷火龙
```

### 双阶段进化

```
绿毛虫 → 铁甲蛹 → 巴大蝶
```

### 特殊进化形态

```
皮卡丘 → (携带雷之石) → 雷伊布
```

| 进化等级 | 数量 | 代表精灵 |

| 7-10 | 较多 | 虫系 |

## 查询技巧

- 进化链列表：https://pokeapi.co/api/v2/evolution-chain?limit=1000
- 按精灵查询：查询精灵数据中的evolutions字段
- 进化石列表：https://pokeapi.co/api/v2/item-category/evolution-stone

## 进化要点

1. **经验累积**：进化前多打怪升级
2. **技能遗忘**：进化后可能遗忘技能，需注意
3. **道具准备**：提前准备进化道具
4. **形态变化**：部分进化会改变属性
